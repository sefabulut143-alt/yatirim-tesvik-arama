YATIRIM TEŞVİK BELGESİ ARAMA UYGULAMASI
==========================================

📁 KURULUM:
- Bu klasörü istediğiniz yere kopyalayabilirsiniz
- Hiçbir kütüphane yüklemenize gerek yok
- Sadece Windows işletim sistemi yeterli

🚀 ÇALIŞTIRMA:
- "Yatırım Teşvik Belgesi Arama.exe" dosyasına çift tıklayın
- Uygulama otomatik olarak açılacak

💡 ÖZELLİKLER:
- Firma adına göre arama
- Açık/Kapalı belge durumu gösterimi
- Detaylı belge bilgileri
- Kopyala-yapıştır özelliği
- Açık/Koyu tema seçeneği
- Hızlı açılış

📋 KULLANIM:
1. Firma adını büyük harfle yazın
2. "Ara" butonuna tıklayın
3. Sonuçları inceleyin
4. İstediğiniz bilgiyi kopyalayın

⚠️ NOT:
- Uygulama ilk açılışta veri dosyasını yükler
- Bu işlem birkaç saniye sürebilir
- Sonraki açılışlarda çok hızlı açılır

🔧 TEKNİK BİLGİLER:
- Python ile geliştirilmiştir
- PyInstaller ile exe'ye çevrilmiştir
- Tamamen bağımsız çalışır
- Windows 10/11 uyumludur
